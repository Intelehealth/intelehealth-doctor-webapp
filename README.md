### Installing
A step by step series of examples that tell you how to get a development environment running
1. Clone or download this repository
2. Install all the dependencies.
```
"npm install"
```    
3. Start the server
```
"ng serve"
```

4. Open in browser
```
 "localhost:4200"
```
